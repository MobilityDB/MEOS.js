Enumeration: TemporalType ​

Defined in: core/types/temporal/Temporal.ts:105

Subtype of a temporal value.

A temporal can be stored as one of four subtypes depending on the WKT string passed to the constructor: Instant -> "true@2001-01-01" InstantSet -> "{true@2001-01-01, false@2001-01-02}" (discrete) Sequence -> "[true@2001-01-01, false@2001-01-02]" SequenceSet -> "{[true@2001-01-01, false@2001-01-02], [true@2001-01-03]}"
