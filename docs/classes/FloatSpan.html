    Class FloatSpan

    A contiguous range of IEEE 754 double-precision floats.

    Example

    const s = FloatSpan.fromBounds(1.5, 10.5); // [1.5, 10.5)
    console.log(s.lower()); // 1.5
    console.log(s.width()); // 9.0
    s.free();
