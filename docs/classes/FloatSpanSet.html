    Class FloatSpanSet

    An ordered set of disjoint FloatSpan values.

    Example

    const ss = FloatSpanSet.fromString('{[1.5, 5.5), [8.0, 12.0)}');
    console.log(ss.numSpans()); // 2
    console.log(ss.lower());    // 1.5
    console.log(ss.upper());    // 12.0
    ss.free();
